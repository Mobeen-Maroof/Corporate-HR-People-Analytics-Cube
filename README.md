# Corporate-HR-People-Analytics-Cube

## Overview

This project is developed as part of the M-Tech Data Science Internship.

The objective of this project is to clean and deduplicate a messy employee HR database using Python and Pandas, ensuring accurate and consistent employee records. It also aims to perform HR analytics by engineering useful features such as employee tenure and attrition riskandvisualizing key insights through charts and dashboards for better decision-making.

---

## Features

- Load employee dataset from Excel
- Detect and remove duplicate employee records
- Standardize employee names
- Handle missing values
- Perform Exploratory Data Analysis (EDA)
- Generate HR analytics and insights
- Visualize employee data using charts
- Display employee records in a GUI table
- Save cleaned dataset to Excel
- User-friendly PyQt5 GUI

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- PyQt5
- OpenPyXL
- Jupyter Notebook

---

## Dataset

The dataset contains employee information, including:

- Employee ID
- Employee Name
- Department
- Salary
- Email Address
- Joining Date
- Status
The sample dataset includes duplicate records and inconsistent formatting for demonstrating data cleaning and HR analytics.

---

## Project Workflow

1. Load the employee dataset
2. Explore the data (EDA)
3. Detect duplicate records
4. Clean and standardize employee data
5. Analyze HR metrics
6. Create visualizations
7. Save the cleaned dataset
8. Display results using the PyQt5 GUI
